Feb 12, 2026

v1.0.0

Stable Release

The first public stable release of WebDock Ultimate. Turn any website into a desktop app.

  • Native Windows: WebDock now compiles to native .exe wrappers.
  • Local Server: Built-in Python engine to host local HTML folders.
  • Dark Mode: System-sync dark mode support.
  • Fix: Resolved memory leak in background tabs.
